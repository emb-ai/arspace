# Размещение на хостинге Рег.ру

Данная инструкция описывает, как разместить arspace на хостинге [рег.ру](https://www.reg.ru).

Более подробную инструкцию можно найти на [официальном сайте](https://help.reg.ru/support/hosting/razmeshcheniye-sayta-otobrazheniye-v-brauzere/kak-razmestit-gotovyy-sayt-v-internete#1).

## 1. Регистрация домена

Зайдите в личный кабинет после регистрации. Вы попадете на "главную" страницу, в окне "Поиск нового домена" введите желаемое имя сайта (например, `arspace.ru`). Осталось только подобрать незанятое имя (или выкупить занятое). После регистрации ваш домен будет отображаться в личном кабинете в разделе "Домены".

## 2. Настройка хостинга

Вам нужно приобрести место, где будет "лежать" ваш сайт — [хостинг](https://www.reg.ru/hosting/). Для этого можете приобрести самую стандартную подписку "Host-0".

> **Важно: SSL-сертификат**  
> Вашему сайту нужен будет доступ к камере телефона пользователя. Без SSL-сертификата это вряд ли получится — телефон будет блокировать запрашиваемый сайтом доступ к камере. Поэтому вместе с хостингом стоит приобрести SSL-сертификат для сайта (на рег.ру эта функция уже идет в комплекте с хостингом).

## 3. Привязка домена к хостингу

Привязать домен к хостингу — значит прописать для домена DNS-серверы. DNS-серверы устанавливают связь между именем сайта и IP-адресом сервера, на котором размещен этот сайт. Этот процесс может занять до 24 часов.

Для этого зайдите в **Личный кабинет → Домены**, выберите нужный домен и нажмите "Подключить" в колонке "Подробности".

## 4. Размещение сайта на хостинге

Самый простой путь — размещение сайта через панель управления хостингом:

1. **Хостинги → Хостинг Host-0 → Панель управления ispmanager**
2. В ispmanager перейдите в **Сайты → галочка у нужного → Файлы сайта**
3. Загрузите все файлы из этого репозитория в открывшийся корневой каталог

> **Примечание об обновлении файлов**  
> Иногда если вы удаляете файл, а потом загружаете новый (касается .html / .js файлов), они не обновляются, а вызываются старые. Для обновления нужно либо переименовать файл в новый (что не очень удобно), либо нажать на файл, который вы хотите поменять, откроется текстовый редактор и там уже можно изменить код файла на нужный.

## 5. Проверка

Когда ваш сайт готов и хостится, зайдите на него, чтобы убедиться, что все анимации работают.

**Важно:** Используйте `https://` префикс при заходе на сайт (например, `https://arspace.ru`), так как некоторые телефоны блокируют использование камеры на незащищенных сайтах.
